# Zomato Analysis
## Project-Objective
- To analyze Zomato restaurant data to uncover trends in customer preferences, pricing, and restaurant performance.
To clean, transform, and visualize the dataset for meaningful insights that support data-driven decision-making.
To create clear dashboards and analytical reports that help stakeholders understand key KPIs and business patterns.

## Process
- Imported and understood the Zomato dataset, identifying key fields like ratings, cuisines, cost, and location.
- Cleaned and prepared the data by removing duplicates, handling missing values, and standardizing formats.
- Performed exploratory data analysis (EDA) to identify trends in ratings, pricing, cuisines, and city-wise performance.
- Created visualizations and dashboards to represent key KPIs such as average rating, cost for two, and cuisine popularity.
- Derived insights and recommendations based on the analysis to support data-driven decision-making.
